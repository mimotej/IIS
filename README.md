# Projekt IIS

Projekt je spoštěn pomocí docker
Pro vytvoření containeru a jeho spuštění je třeba mít docker (např ubuntu: sudo apt install docker.io a sudo apt install docker-compose) 
Následně spuštění:
```docker-compose up```
Web je na http://0.0.0.0/5000
